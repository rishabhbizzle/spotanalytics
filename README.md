
# SpotAnalytics

One Stop Destination for Spotify Streaming Records and Data of Your Favourite Artists.

![ss](https://github.com/rishabhbizzle/spotanalytics/assets/100997409/782551a9-7683-4731-8022-d9c476d55bcd)



## Project Links

 - [Live Link](https://statsforfans.azurewebsites.net/)
 

# Features


### -> Extensive Artists Streaming Data (like total and daily streams of their songs and albums)
### -> Updated Most Streamed Artists Lists (gets updated daily)
### -> Updated Most Streamed Songs Lists (gets updated daily)
### ->  Realtime details about the current Most popular artists on the app
### ->  Users can request streaming data for their fav artists if they are not included as of now

## Tech Stack

**Client:** HTML/CSS, Javascript, BootStrapCSS

**Server:** Django, BeautifulSoup4, Pandas, Azure
